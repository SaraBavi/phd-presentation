Seam Path Detection

www.vetrontypical-europe.com Details: Seam_Path_Detection.pdf

Hardware & Integration

Goal: Detect seam paths on fabrics to adjust decorative stitch positions automatically.
Setup: Industrial Sewing Machine, XIMEA MONO8 Camera, and CAN protocol data exchange.

System Dynamics

Achieving continuous tracking to minimize manual alignment and human error.
Key challenges in precise curve detection include motion-induced vibrations and fabric deformation (wrinkling).

Algorithm & Path Detection

Vision Pipeline: CLAHE, Gaussian Blur, and Gabor Filters extract texture patterns.
Optimal Control: Dynamic programming finds the lowest-cost seam in a bidirectional cumulative cost matrix.
Path Fitting: Polynomial curve fitting combined with RANSAC ensures smooth control.
